Welcome to GitHub Pages.

This page contains my progress on my R code for my PhD project. Current code is made to run simulations of PLS variable selection algorithms with mixed data types.

Authors and Contributors

The content of the codes is based on some of the existing R packages. The modifications are made so that different algorithms can be linked together to handle both categorical and numerical codes. Below is a list of Authors who created some of the original functions: 1. Dr. Benoit Liquet 2. Dr. Beaton Derek 3. Dr. Russolillo, Giorgio 4. Dr. Le Cao

A special thanks is given to Dr. Yoni Nazarathy for his contribution on the correction of mathematical theory of the code.
